# Wilderness Escape - Tree Data Structure
> A game where the user gets to choose at each step how to continue the story. This creates a different experience depending on the options they chose.

## General info
Use the tree data structure to keep track of the different paths a user may choose in the game.

## Technologies
* Python 3.7.6 
* Git Bash 2.27.0
* Atom v1.49.0
* Windows 10 Pro

## Features
* A TreeNode class that keeps track of the current portion of the story and the available choices at current point.
  * The instances of the Treenode class represent the different pieces of the story.
  * Method to add new choices to a story piece.
* The game is played by travesing the Tree from the first story piece (root) until the last.
  * User input is taken to choose between 2 options to continue the story. This repeats until there are not more story pieces left.
