# Hacking the Fender - Files Programming
> Retrieve a list of stolen usernames and passwords from a hacker.

## General info
Practice working with Python file objects to read and write .txt, .csv and .json files.

## Technologies
* Python 3.7.6 
  * csv module
  * json module
* Git Bash 2.27.0
* Atom v1.49.0
* Windows 10 Pro

## Features
* Read the file containing the stolen passwords using DictReader.
* Create a list of compromised usernames.
* Create a .json file using json.dump that contains a message about the successful retrieval of the usernames.
* Replace the file containing the the stolen passwords with one containing a signature.
