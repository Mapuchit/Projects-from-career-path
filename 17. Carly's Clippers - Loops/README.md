# Carly's Clippers - Loops
> Analyze the data collected from the operation of a hair salon.

## General info
Practice loops in Python, including range and list comprehension.

## Technologies
* Python 3.7.6
* Git Bash 2.27.0
* Atom v1.49.0
* Windows 10 Pro

## Features
* Find the average price of a haircut offered by the salon.
* Cut the prices of all haircuts by a certain amount to generate new discounted prices.
* Calculate the total revenue generated for a given week.
* Find the average daily revenue.
* Find all the haircuts under a certain price.
