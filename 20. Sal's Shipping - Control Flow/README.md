# Sal's Shipping - Control Flow
> A program that helps a shipping company to determine the cheapest way to ship a package of given weight.

## General info
Practice control flow and writing functions in Python. 

## Technologies
* Python 3.7.6
* Git Bash 2.27.0
* Atom v1.49.0
* Windows 10 Pro

## Features
* Choice of 3 different shipping types.
* Calculate cheapest shipping type for parcel weight.
* Inform user of cheapest shipping type and price.
