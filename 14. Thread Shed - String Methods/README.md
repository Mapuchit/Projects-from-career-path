# Thread Shed - String Methods
> Clean up and analyze sales data of a sewing hobby shop stored in single string.

## General info
Practice Python string methods to clean up data. Use split, strip and list methods.

## Technologies
* Python 3.7.6
* Git Bash 2.27.0
* Atom v1.49.0
* Windows 10 Pro

## Features
* Determine the total value of the days sales.
* Determine the number of sales during the day
* Keep track of the names of the customers during the day.
* Find how much thread of any specific color was sold.
