# Abruptly Goblins - Dictionaries
> Find the best date for a tabletop game night in a comic book store using an automated selection system.

## General info
Use Python dictionaries to keep track of gamers availability on different days of the week.

## Technologies
* Python 3.7.6
* Git Bash 2.27.0
* Atom v1.49.0
* Jupyter Notebook 6.0.3
* Windows 10 Pro

## Features
* Add a gamer and his/her availablity to the list of registered gamers.
* Use a frequency table to map each day of the week with gamer availability.
* Find the day of the week on which the most number of players are available.
* Generate emails to all gamers who are available on the chosen day to notify them of the game night.
* Select a second game night by finding the best day for all players who cannot attend the first game night.
