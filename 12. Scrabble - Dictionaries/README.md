# Scrabble - Python Dictionaries
> Processing data generated by a group of friends playing scrabble.

## General info
Using dictionaries to organize players, words and points.

## Technologies
* Python 3.7.6
* Git Bash 2.27.0
* Atom v1.49.0
* Windows 10 Pro

## Features
* Map letters to points as key:value pairs of a dictionary, includes blank tiles.
* Function that takes in a word and returns how many points that word is worth. Can handle both upper and lower case characters.
* Keep track of the list of the words each player has played and their total points.
* Functionality to play a word and update the points table.
* Display the most recent word played, player name and updated score.
