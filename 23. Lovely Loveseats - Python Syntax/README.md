# Lovely Loveseats - Python Syntax
> A small project on Python syntax from the Computer Science Career Path on [Codecademy](https://www.codecademy.com/learn).

## General info
Creating receipts for customers in a furniture store.

## Technologies
* Python 3.7.6
* Git Bash 2.27.0
* Atom v1.49.0
* Windows 10 Pro

## Features
* Store names and prices of items in variables.
* Print the names and descriptions of items purchased by a customer, using string concatenation.
* Calculate the total price of all items purchased by a customer.
* Add a percentage of sales tax to the total price.
* Print the total price of items including sales tax.
