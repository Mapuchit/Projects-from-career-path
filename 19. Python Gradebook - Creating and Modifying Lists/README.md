# Python Gradebook - Creating and Modifying Lists
> A program that organizes a students grades and subjects in a gradebook.

## General info
Practice operations on lists in Python. 

## Technologies
* Python 3.7.6
* Git Bash 2.27.0
* Atom v1.49.0
* Windows 10 Pro

## Features
* Add subject and corresponding grade to individual "grades" and "subjects" lists.
* Combine "grades" and "subjects" lists into "gradebook" using the zip operation.
* Add a subject and it's corresponding grade to the "gradebook" list.
* Extend "gradebook" to include last years grades.
