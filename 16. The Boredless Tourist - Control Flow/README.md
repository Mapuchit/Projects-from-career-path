# The Boredless Tourist - Control Flow
> A recommendation engine that matches the users interests with available attractions at a given tourist destination.

## General info
Practice Python control flow, lists and loops.

## Technologies
* Python 3.7.6
* Git Bash 2.27.0
* Atom v1.49.0
* Windows 10 Pro

## Features
* Find traveler destination in internal database (list).
* Add new attraction to a destination.
* Use tags to identify type of attraction.
* Match travelers interests to attraction tags to find best attraction at destination.
* Print message to inform traveler of recommended attractions.
