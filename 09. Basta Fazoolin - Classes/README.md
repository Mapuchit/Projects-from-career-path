# Basta Fazoolin - Classes
> Organize the business operations of an italian restaurant franchise.

## General info
Use Python classes to manage the available menus at the different restaurant locations.

## Technologies
* Python 3.7.6 
* Git Bash 2.27.0
* Atom v1.49.0
* Windows 10 Pro

## Features
* A Menu class that keeps track of food items, prices and time of availability for the different menus.
* Calculate bill and display total price.
* A Franchise class that keeps track of the address of the restaurant and the menus they offer.
* Display the menus available at a given restaurant at a selected time of day.
* A Business class that keeps track of the name of the business and it's franchises.
* Create new business with different franchises and menus.
