# Coded Correspondence - String Methods
> Send secret messages to a friend using different ciphers.

## General info
Practice Python string methods to encode and decode secret messages.

## Technologies
* Python 3.7.6
* Git Bash 2.27.0
* Atom v1.49.0
* Jupyter Notebook 6.0.3
* Windows 10 Pro

## Features
* Encode and Decode messages using the Caesar Cipher.
* Coder and Decoder functions that work with variable offset value to automate the process.
* Decode message with unknown offset value.

* Functions that can encode and decode a message using a given keyword.
