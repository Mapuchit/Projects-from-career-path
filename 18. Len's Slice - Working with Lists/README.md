# Len's Slice - Working with Lists
> Organizing sales data for a pizza joint.

## General info
Practice operations on lists in Python, including zip, sort, count and list slicing.

## Technologies
* Python 3.7.6
* Git Bash 2.27.0
* Atom v1.49.0
* Windows 10 Pro

## Features
* Keep track of types of pizzas sold and their prices.
* Sort pizzas by price.
* Count the number of different pizzas available for a given price.
